# Contributing / Contribuindo

## English

- Run `npm run prettier` before opening a PR.
- Run `npm run lint` to catch code issues.
- Run `npm run prettier:check` to verify formatting in CI mode.
- Run `npm test` against a default Salesforce org before opening a PR.
- Keep commits small and focused.
- Use clear commit messages.
- Update the README when behavior changes.

## Português

- Execute `npm run prettier` antes de abrir um PR.
- Execute `npm run lint` para encontrar problemas de código.
- Execute `npm run prettier:check` para verificar formatação em modo CI.
- Execute `npm test` contra uma org Salesforce padrão antes de abrir um PR.
- Mantenha commits pequenos e focados.
- Use mensagens de commit claras.
- Atualize o README quando o comportamento mudar.
